# BR25

- Series: **AC696N** and **AC636N**, likely **AC608N** too
- CPU Core: [pi32v2](../../cpu/index.md#pi32)
- SRAM: 153.5k in total:
  * 128k of RAM0
  * 8k of RAM1
  * some more extra 1.5k following RAM1
  * 16k of icache RAM
- [MaskROM](maskrom.md): 32k
- Additional info:
  * [I/O function map](iomap.md)
  * [IRQ map](irq.md)
  * [Memory map](memmap.md)
